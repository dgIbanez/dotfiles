## Configuration

Copy this folder to `~/.config` directory.

I use Lasy as packet manager, so the plugins insalation and updates are automatic.



## Plugins desciption and dependencies:

### plenary

**Repo:** [nvim-lua/plenary.nvim](https://github.com/nvim-lua/plenary.nvim) <br>
**Description:** <br>
This plugin has many common lua functions and it a dependency for others plugins too.

---

### colorscheme

**Repo:** [bluz71/vim-nightfly-guicolors](https://github.com/bluz71/vim-nightfly-colors) <br>
**Description:** <br>
As its name suggests, it's the colorscheme ;)

---

### gitsigns
**Repo:** [lewis6991/gitsigns.nvim](https://github.com/lewis6991/gitsigns.nvim) <br>
**Description:** <br>
Add differences and status signs of git to the side bar.

---

### lualine

**Repo:** [nvim-lualine/lualine.nvim](https://github.com/nvim-lualine/lualine.nvim) <br>
**Description:**<br>
Change the boring status bar of nvim for anohter totally awesome.(and more descriptive)
